# Mean, Spread, and the Extremes

Static web report explaining how differences in mean and standard deviation can have amplified effects in the tails of normal distributions.

The page is a standalone HTML report generated from a local curated gallery of men/women distribution examples. It uses normal-curve summaries as compact visual models, with source notes and caveats where the source data are top-tail, self-report, population indicators, or scale scores.

Public page: <https://davidjpramsay.github.io/mean-spread-extremes-report/>
